# Git 101 Web

An interactive web app for learning Git through hands-on lessons. Work through bite-sized exercises covering everything from basic commits to branching, merging, and beyond — all in the browser, no terminal required.

## Try It Out

**Live site:** [https://itsdingze.github.io/git-101-web/](https://itsdingze.github.io/git-101-web/)

Or run it locally:

```bash
git clone https://github.com/itsdingze/git-101-web.git
cd git-101-web
npm install
npm run dev
```

## Tech Stack

React + TypeScript, Vite, Tailwind CSS, Zustand, Framer Motion, and dnd-kit.
